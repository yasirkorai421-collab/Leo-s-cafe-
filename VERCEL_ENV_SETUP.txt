=================================================================================
  AFTER DEPLOYMENT
=================================================================================

1. 🎨 **SETUP CLOUDINARY FIRST** - Images won't work with demo credentials!
   - Follow instructions in CLOUDINARY_SETUP.md
   - Get free account: https://cloudinary.com/users/register_free
   - Update the 3 CLOUDINARY variables in Vercel Settings
   - Redeploy after updating

2. 🔒 **ENABLE ROW LEVEL SECURITY (CRITICAL)** - Database is exposed without RLS!
   - Follow instructions in docs/RLS_SECURITY.md
   - Run: scripts/setup-rls.sh
   - Or apply prisma/migrations/add_rls_policies.sql in Supabase SQL Editor
   - Verify with test queries in documentation
   - ⚠️  WITHOUT RLS, USERS CAN ACCESS ALL DATA

3. Update NEXT_PUBLIC_APP_URL with your actual Vercel URL

4. Create admin account: npm run admin:create

5. Visit: https://your-url.vercel.app/admin/login
